📋 Como funciona
1

Escolha o HD externo

Na tela "Mover para HD", selecione qual disco externo vai receber os programas.

2

Selecione os programas

Veja os programas instalados no SSD, selecione quais mover. Jogos grandes são os melhores candidatos.

3

Mova com um clique

O programa copia tudo para o HD, cria um atalho inteligente (Junction) no lugar original. O Windows não percebe diferença — o programa continua funcionando normalmente.

4

Novos programas direto no HD

Na aba "Instalar no HD", redirecione a pasta padrão do Windows para o HD externo.

⚠️ Atenção antes de começar
  • Execute o programa como Administrador (necessário para criar Junctions)
  • O HD externo deve estar conectado sempre que for usar os programas movidos
  • Não desconecte o HD durante a cópia
  • Feche os programas antes de movê-los

📋 O que este recurso faz
  • Altera a pasta padrão Program Files do Windows para apontar para o HD
  • Novos programas instalados pelo .exe padrão irão automaticamente para o HD
  • Steam, Epic e outros launchers continuam com suas próprias configurações
  • Você pode desfazer a qualquer momento pelo botão "Restaurar"
🎮 Jogos no Steam / Epic / GOG

Para jogos via launchers, é mais fácil configurar diretamente no launcher:

Steam: Configurações → Downloads → Pastas da Biblioteca Steam → Adicionar pasta no HD
Epic Games: Configurações → Gerenciar Jogos → Alterar pasta de instalação
GOG Galaxy: Configurações → Padrões para Instalação → Mudar diretório
